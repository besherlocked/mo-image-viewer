# Mo图 - 跨平台看图软件

一款简洁高效的跨平台图片查看器，支持 macOS 和 Windows。

## 功能特性

- **多格式支持**: JPG, PNG, GIF, BMP, WebP, TIFF, SVG, AVIF, ICO, HEIC
- **窗口模式**: 全屏/窗口化切换，窗口可自由缩放，图片自适应
- **图片缩放**: 鼠标滚轮缩放，以鼠标位置为中心
- **图片导航**: 左右方向键翻页，上下方向键切换文件夹
- **长按加速**: 长按翻页键，5秒内逐渐加速到最快
- **文件夹穿透**: 可选自动进入同目录下的其他文件夹
- **图片预加载**: 后台预加载前后各3张图片，翻页无卡顿
- **拖放打开**: 拖放图片或文件夹到窗口直接打开
- **迷你地图**: 放大后右下角显示全局预览，可拖拽定位
- **多适配模式**: 适应窗口/宽度/高度/原始大小
- **旋转翻转**: 支持90°旋转和水平/垂直翻转
- **背景切换**: 深色/浅色/灰色/棋盘格背景
- **自定义快捷键**: 所有操作快捷键可自定义
- **文件关联**: 可注册为系统默认图片查看器
- **侧边栏**: 左边缘悬停显示文件夹列表和缩略图预览

## 环境要求

- Node.js >= 18
- Rust >= 1.70
- 系统依赖 (详见 [Tauri 前置条件](https://v2.tauri.app/start/prerequisites/))

## 开发

```bash
# 安装依赖
npm install

# 开发模式运行
npm run tauri dev

# 构建发布包
npm run tauri build
```

## 快捷键

| 功能 | 默认快捷键 |
|------|-----------|
| 上一张图片 | ← |
| 下一张图片 | → |
| 上一个文件夹 | ↑ |
| 下一个文件夹 | ↓ |
| 重置缩放 | 0 |
| 顺时针旋转 | R |
| 逆时针旋转 | Shift+R |
| 水平翻转 | H |
| 垂直翻转 | V |
| 全屏切换 | F11 / Cmd+Enter |
| 缩放 | 鼠标滚轮 |

## 技术栈

- **框架**: Tauri v2 (Rust + Web)
- **前端**: React 18 + TypeScript + Tailwind CSS
- **状态管理**: Zustand
- **图片处理**: Rust image crate
